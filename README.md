# Install
* Available at localhost:3000
* Image upload functionality provided by Amazon AWS S3, Amazon AWS Access Key and Secret Keys along with a bucket titled "casino-chic" is required to use image upload locally.

```sh
$ git clone
$ cd casino-chic
$ npm install
$ npm start
```
